## 💡 새롭게 배운 점

### FAST API
FastAPI는 파이썬으로 “API 서버”를 만드는 프레임워크
공식 문서도 FastAPI를 파이썬 타입 힌트를 기반으로 API를 만들기 위한 빠른 웹 프레임워크

#### 흐름
- 프론트가 백엔드에 요청
- 백엔드가 AI 서버(FastAPI)에 요청
- AI 서버가 모델 추론
- 결과를 백엔드에 반환
- 백엔드가 프론트에 최종 응답

```bash
[프론트]
   ↓
[백엔드 서버]
   ↓ HTTP 요청
[AI 서버 (FastAPI)]
   ↓
[AI 모델 추론]
   ↓
[JSON 결과 반환]
```
