## 💡 새롭게 배운 점

### 예외 처리란?

예외 처리(exception handling)란, 프로그래머가 예기치 못한 예외의 발생에 미리 대처하는 코드를 작성하는 것

### 사용 방법

- 파일이 없거나 잘못된 파일명을 프로그램 내에서 접근한다면?
- 프로그래머의 실수로 배열의 인덱스 값을 넘어가버리는 코드를 작성한다면?
- 나눗셈 계산을 할 때 N / 0 같은 코드를 작성한다면? 
- 기타 등등 프로그래머가 예상치 못한 경우의 에러가 발생 할 경우에 이 상황을 대체 할 수 있는 코드가 필요함

### 예외 처리 예시

- Throw 문 사용 (or return 문) (자바 스크립트 예시)

- 에러 체크 후 바로 함수를 바로 빠져나가는 방법을 즉시탈출이라 한다
- 에러 체크 후에 본 코딩을 하는 습관을 가지기 
- Throw가 발생되면, 해당 인자를 call stack (호출 함수 stack)을 찾아 바로 위의 호출 함수에게 알림 
- 로그와 함께하면 더욱 쉽게 에러 파악 가능
