## 블록 체인의 이해
혜린님이 정리해준 내용이 이해하기 쉬워서 인용해왔다.

### 체인 / 코인 / 토큰의 개념
1. 체인 : 하나의 시스템, 혹은 운영체제 (ex. 이더리움, 비트코인, 폴리곤, 솔라나 등)

2. 코인 : 우리가 흔히 아는 암호 화폐. 각 체인마다 고유한 코인을 갖고 있음. 즉 체인이 없으면 코인이 존재할 수 없다 (ex. 이더리움 -> ETH / 비트코인 -> BTC / 솔라나 -> SOL)

3. 토큰 : 체인 위에서 스마트 컨트랙트로 만들어진 자산. 체인 운영자나 개발자 등이 만드는 게 아니고, 해당 체인을 활용해서 모두가 생성 가능. NFT가 여기에 해당됨.

++ 참고로 우리가 체인을 개발하지 않는 이상, 저희가 생성하는 NFT나 스테이블 코인은 전부 토큰의 특성을 가진다.

```
(예시)
체인 : 동물의 숲
코인 : 동물의 숲에서 사용되는 고유 화폐인 >> 벨 <<
토큰 : 동숲 내 아이템 정도(비록 동숲 유저는 아이템을 만들 수 없지만,,,,) 거래는 할 수 있음
```


### AI 강의 LLM 작동방식에서의 포인트

```
┌───────┐   ┌──────────┐   ┌──────────────────┐   
│ Input │ → │ Tokenize │ → │ Token Embedding  │ → 
└───────┘   └──────────┘   └──────────────────┘   
┌──────────┐   ┌─────┐   ┌───────────────┐   ┌────────────────────┐
│ Encoding │ → │ LLM │ → │ Output (Logit)│ → │ Sampling & Decoding│
└──────────┘   └─────┘   └───────────────┘   └────────────────────┘
```


- LLM의 output은 `문장`이 아니라 `logit`이다. 즉, `Raw Probability`가 나온다. 매 순간마다 분류를 하고 있음.
